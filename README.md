#NoDelayApp — Sistema de Pedido e Retirada Rápida para Cantinas

O NoDelayApp é um sistema web desenvolvido para reduzir filas em cantinas de escolas e universidades.  
O projeto permite que alunos realizem pedidos online, simulem pagamento, acompanhem o status da preparação e recebam notificações em tempo real.  
Os operadores da cantina podem visualizar pedidos, atualizar status e gerenciar estoque e preços dos salgados.

Este projeto foi desenvolvido como parte das atividades acadêmicas da faculdade, aplicando conhecimentos de HTML, CSS, Java e Banco de Dados MySQL.

Objetivo Geral do Projeto

Criar uma aplicação que implemente um fluxo completo de “Peça e Retire”, oferecendo:

- Cadastro e login de alunos  
- Login exclusivo para operador
- Listagem de produtos com preço e quantidade disponível  
- Carrinho de compras  
- Simulação de pagamento  
- Criação de pedidos com desconto automático do estoque  
- Notificações de status:  
•	Pedido recebido  
•	Em preparação  
•	Pronto para retirada  
•	Finalizado  
- Painel do operador com controle total dos pedidos  
- Ajuste de estoque e preço dos salgados
